Keyboard Shortcuts

Actions

  • o                open file at hunk
  • l                stage line / unstage line (in cached mode)
  • h                stage hunk / unstage line (in cached mode)
  • L                reset line
  • H                reset hunk
  • .                go to next hunk
  • ,                go to previous hunk

Other

  • ?                show this help popup
  • {super_key}-z          undo last action

Vintageous friendly mode

  • j                go to next hunk
  • k                go to previous hunk

Settings

  • {super_key}-,      Change Settings for current syntax
