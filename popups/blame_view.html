Keyboard Shortcuts

`Enter` to show all commands

Actions

  • Open Commit                           Open the commit where the cursor is
  • Blame before selected commit          Checkout this file at the commit before the commit where the cursor is
  • Blame on one older commit
  • Blame on one newer commit
  • Show file at most recent commit       Show file at the commit
  • Show file at this chunk's commit      Show file at the commit where the cursor is

Keyboard Shortcuts

  • ,                go to next chunk
  • .                go to previous chunk
  • <                Blame on one older commit
  • >                Blame on one newer commit
  • alt+<            Blame before selected commit
  • alt+>            Blame on one newer commit
  • ?                show this help popup

Vintageous friendly mode

  • j                go to next chunk
  • k                go to previous chunk
